Package ogcserver

Mapnik OGC WMS Server.

Submodules
  • mapnik.ogcserver.WMS: Interface for registering map styles and layers for availability in WMS Requests.
  • mapnik.ogcserver.cgiserver: CGI/FastCGI handler for Mapnik OGC WMS Server.
  • mapnik.ogcserver.common: Core OGCServer classes and functions.
  • mapnik.ogcserver.configparser: Change SafeConfigParser behavior to treat options without values as non-existent.
  • mapnik.ogcserver.exceptions: Custom OGCServer Exceptions
  • mapnik.ogcserver.modserver: Mod_python handler for Mapnik OGC WMS Server.
  • mapnik.ogcserver.wms111: WMS 1.1.1 compliant GetCapabilities, GetMap, GetFeatureInfo, and Exceptions interface.
  • mapnik.ogcserver.wms130: WMS 1.3.0 compliant GetCapabilities, GetMap, GetFeatureInfo, and Exceptions interface.
  • mapnik.ogcserver.wsgi: WSGI application wrapper for Mapnik OGC WMS Server.
